# Brand

Apple-simple. Four colours. That's it.

---

## Colours

| Token     | Hex       | Role                                       |
| --------- | --------- | ------------------------------------------ |
| `ink`     | `#0a0a0f` | Background. Near-black, faintly blue.      |
| `surface` | `#12121a` | Anything raised.                           |
| `mint`    | `#8aff80` | The one accent.                            |
| `purple`  | `#7e64ff` | Mint's hover, occasional gradient partner. |

Text is white, dialled down by opacity. No mid-greys.

## Type

Apple system fonts — `SF Pro Display` and `SF Pro Text`, `Inter` as fallback. Nothing loads over the network.
